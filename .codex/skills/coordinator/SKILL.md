# Coordinator Skill

Use this skill to route tasks:
- `research-worker`: market/product research summaries
- `backend-builder`: API, DB, connector logic
- `frontend-builder`: UI/UX and integration
- `security-reviewer`: secrets, auth, data safety checks
- `release-publisher`: docs/site/readme sync and publish checks

Always require:
1. clear output artifact,
2. verification command,
3. next-step handoff.
