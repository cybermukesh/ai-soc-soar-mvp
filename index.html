Core Stack

FastAPI, SQLite (Postgres-ready), React/Vite, Wazuh/OpenSearch, Shuffle-first SOAR plan with n8n optional.

Governance

Clean-room policy, audit logging, RBAC controls, masked secrets, connector status history.

High-Level Architecture

  • Wazuh/OpenSearch ingestion through connector layer.
  • FastAPI backend for auth, RBAC, connectors, settings, triage, incidents, and audit logs.
  • Normalized alert schema and low-token AI triage engine with cache strategy.
  • Signal/noise scoring, duplicate grouping, suppression reason, and analyst queue routing.
  • Incident lifecycle with timeline events and analyst action controls.
  • SOAR dispatch foundation added for n8n test webhooks with persisted run logs and admin approval gates for containment actions.
  • React dashboard for operators, admin controls, AI provider settings, and visibility metrics.
